# Workflow Contract

This contract freezes the newsroom workflow used by Bassac Post.

## Roles

- writer
- editor
- translator
- admin

## Workflow States

1. Draft
2. Submitted
3. Editor Approved Source
4. In Translation
5. Translation Submitted
6. Editor Approved Translation
7. Admin Final Review
8. Published

## Non-Negotiable Rules

- Translator can submit translation work.
- Writer can submit translation work for own article workflow.
- Translator cannot publish.
- Admin is the final approver before publish.
